Demos

  • Transferable Objects
    Demonstrates the performance gain from using webkitPostMessage(), which uses transferable objects.
  • MediaSource API
    Uses the MediaSource API and the File APIs to split a movie into chunks and 'stream' each chunk to a <video> element.
  • Fullscreen API
    Using the Fullscreen API to take a content <div> into the browser's full screen mode, programmatically.
  • Saving named file downloads, client-side
    Using window.URL.createObjectURL() to create a file and the download attribute to specify it's a download.
  • navigator.onLine
    Using navigator.onLine and online/offline events to check for connectivity.
  • Generating URIs to File Resources
    Using window.URL.createObjectURL(), FileEntry.toURL(), and dataURLs with the FileSystem API.
  • Background News [ install ]
    Using Google Chrome 10's background feature.
  • Book of the Day [ install ]
    Using Google Chrome 10's background feature.
  • Clock
    A clock implemented using <canvas> and Server-Sent Events
